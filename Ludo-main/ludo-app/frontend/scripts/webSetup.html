3Create Admin User

Set up your admin user:

  1. Go to Firebase Console ‚Üí Authentication
  2. Find your admin user or create one
  3. Copy the User UID
  4. Go to Firestore Database
  5. Create collection: "admins"
  6. Create document with ID = User UID
  7. Add these fields:
{ "role": "admin", "email": "admin@yourdomain.com", "createdAt": "2024-01-01T00:00:00.000Z" }

4Deploy Security Rules

Update Firestore security rules:

  1. Go to Firebase Console ‚Üí Firestore Database ‚Üí Rules
  2. Copy content from firestore.rules file in your project
  3. Paste into Firebase Console
  4. Click "Publish"
